# Minido Research

This is a repository for all notes, materials, and resources compiled for learning and understanding the various development frameworks and tools used in the "Minido" project

This is a Work in Progress. Guidelines for what materials can be posted here or how they should be formatted, has not been decided yet. 

Please credit whatever sources you include in any materials contributed to this repository

## Folder structure


* **assets** - includes any static resources such as screenshots, graphics, text documents, etc

* **backend** - includes anything related to backend topics such as database, APIs, relevant architectural patterns, etc

* **devops** - includes anything related to devops topics such as containers, CI/CD, networking, deployment, etc

* **frontend** - includes anything related to frontend topics such as angular, vue, bootstrap, CSS, etc

* **misc** - includes anything that currently does not fit the above mentioned categories
